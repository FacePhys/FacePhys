# FacePhys V3 返回字段说明

[English](response-fields.md) | [简体中文](response-fields.zh-CN.md)

本文列出 `/api/v3/video/process` 响应中常见模块和字段定义。实际返回字段取决于 API Key 开通的 `fieldSet`，未订阅、信号质量不足或计算条件不满足的模块可能不会出现在响应中。

字段说明中的精度为参考值，受视频质量、光照、头部运动、人脸占比、肤色和采集时长影响。血压、血氧、心理、情绪、行为和活体等结果用于趋势和状态参考，不应替代专业医疗设备或临床判断。

## `cardiac` 基础生理

**计费/模块:** 基础生理包 · 20,000 Token/次

心率、信号质量及逐秒心率序列。cardiac.sqi 决定所有生理指标的可信程度，应优先检查。参考误差：静息 ±2 / 说话 ±5 / 运动 ±8 bpm（SQI ≥ 0.65 典型采集条件）。

| 字段路径 | 参考精度 | 说明 |
| --- | --- | --- |
| `cardiac.hr` | ±2/±5/±8 bpm | 平均心率 (bpm)；正常静息 60–100；< 40 极度过缓；40–60 偏低（运动员）；100–120 轻度过速；> 120 明显过速，建议复检 |
| `cardiac.sqi` | — | 信号质量 0–1；≥ 0.65 高质量（psych 可达完整范围）；0.30–0.65 可用；0.15–0.30 可参考（HRV 误差较大）；< 0.15 丢弃。低 SQI 常见原因：逆光、深肤色（V/VI）、头部运动、口罩遮挡 |
| `cardiac.hr_list[]` | ±2/±5/±8 bpm | 逐秒心率序列 [{hr, ts}]，ts 单位秒。趋势上升=焦虑递增；趋势下降=逐渐平静；相邻帧差 > 20 bpm = 信号噪声 |

## `cardiac.hrv` HRV 心率变异性

**计费/模块:** HRV 神经包 · 30,000 Token/次

完整 HRV 指标（时域 + 频域 + Poincaré 散点）。反映自主神经系统状态，是非接触生理计算中信息量最丰富的维度。主要时域指标参考误差：SDNN / RMSSD 静息 ±5 / 说话 ±15 / 运动 ±20 ms；ibi_mean 静息 ±50 / 说话 ±100 / 运动 ±150 ms；频域与几何指标约 ±10%；呼吸率 ±2–5 次/分。

| 字段路径 | 参考精度 | 说明 |
| --- | --- | --- |
| `hrv.ibi_mean` | ±50/±100/±150 ms | NN 间期均值 (ms)；正常 600–1000；低=心率偏快；高=心率偏慢；参考误差：静息 ±50 / 说话 ±100 / 运动 ±150 ms |
| `hrv.sdnn` | ±5/±15/±20 ms | IBI 标准差 (ms)；正常 50–150；低=自主调节受限（压力/疲劳）；高=调节灵活。临床参考（Task Force 1996）：> 50ms 低风险；20–50ms 中风险；< 20ms 高风险；参考误差：静息 ±5 / 说话 ±15 / 运动 ±20 ms |
| `hrv.rmssd` | ±5/±15/±20 ms | 间期差均方根 (ms)；正常 20–80；低=副交感活性低（压力/过劳）；高=副交感活跃（身体放松）；参考误差：静息 ±5 / 说话 ±15 / 运动 ±20 ms |
| `hrv.pnn50` | ±10% | 间期差 > 50ms 百分比；正常 5–45%；低=迷走张力低；高=迷走张力高，松弛 |
| `hrv.pnn20` | ±10% | 间期差 > 20ms 百分比；正常 15–70%；对短段视频更敏感 |
| `hrv.LF` | ±10% | 低频功率 (ms²)；正常 200–2000；0.04–0.15 Hz；交感+副交感混合调控 |
| `hrv.HF` | ±10% | 高频功率 (ms²)；正常 200–2000；0.15–0.40 Hz；纯副交感/迷走活性 |
| `hrv.LF/HF` | ±10% | 交感/副交感比率；正常清醒 1.5–4.0；< 1.0 深度放松；1.0–2.0 偏放松；4.0–8.0 明显压力；> 8.0 高度交感激活 |
| `hrv.TP` | ±10% | 总功率 (ms²)；正常 500–5000；自主神经总能量储备 |
| `hrv.VLF` | ±10% | 极低频功率 (ms²)；0.003–0.04 Hz；与体温/代谢调节相关；正常 50–500；< 50 代谢调节极弱；> 500 在短视频中多为估算误差（< 5 分钟录像精度低，仅供趋势参考） |
| `hrv.sd1` | ±10% | Poincaré 短轴 ≈ RMSSD/√2 (ms)；正常 15–45；< 10 极低副交感活性（高压/过劳）；10–20 偏低（轻度应激）；20–45 正常；> 45 副交感活跃（深度放松/恢复中） |
| `hrv.sd2` | ±10% | Poincaré 长轴 ≈ √(2·SDNN²−½·SD1²) (ms)；正常 50–150；< 40 长期 HRV 极低（自主调节空间狭窄）；40–80 偏低；80–150 正常；> 150 高（训练有素运动员） |
| `hrv.sd1_sd2` | ±10% | SD1/SD2 比率；正常 0.3–0.7；> 0.7 副交感显著主导（深度放松）；0.5–0.7 偏放松；0.3–0.5 自主平衡；< 0.3 交感主导/慢性应激 |
| `hrv.poincare_s` | ±10% | Poincaré 散点面积 = π×SD1×SD2 (ms²)；< 2000 极低自主调节空间（高风险）；2000–5000 偏低；5000–15000 正常；> 15000 高（运动员/高 HRV 人群） |
| `hrv.breathing_rate` | ±2–5 次/分 | 呼吸率 (次/分)；由 rPPG 信号提取，误差 ±2–5 次/分。分段解读：< 8 刻意减缓/深度冥想；8–12 深度放松；12–16 正常清醒静息；16–20 轻度活跃/说话；20–25 轻度紧张/运动后；> 25 明显过速（焦虑/换气过度）。6 次/分（0.1 Hz）时与心率共振（相干呼吸），HF 功率最大化，是心身调节最优呼吸节律 |
| `hrv.median_nn` | ±10% | NN 间期中位数 (ms)；正常 600–1000；< 600 对应 HR > 100（过速）；600–700 HR 86–100（偏快）；700–900 HR 67–86（正常）；> 1000 HR < 60（过缓/运动员）；对偶发异位搏动比 ibi_mean 更稳健 |
| `hrv.cvnn` | ±10% | NN 变异系数（SDNN/ibi_mean）；正常 0.03–0.12；< 0.03 极低节律变化（可能刚性心率，需关注）；0.03–0.06 偏低（应激/疲劳）；0.06–0.12 正常；> 0.15 波动剧烈（高 HRV/信号噪声风险） |
| `hrv.cvsd` | ±10% | 连续差值变异系数（RMSSD/ibi_mean）；正常 0.02–0.10；< 0.02 短期变化极低（副交感抑制/过劳）；0.02–0.06 偏低；0.06–0.10 正常；> 0.10 短期高变化（年轻/放松）；> 0.15 波动剧烈（噪声风险） |
| `hrv.heart_rate` | ±2/±5/±8 bpm | HRV 基础心率 (bpm)；由 NN 间期计算，可能与 cardiac.hr 略有差异 |

## `bp` 非接触血压估算

**计费/模块:** 基础生理包

> 注意：估算值，误差约 ±8–12 mmHg，不可替代医疗设备，供趋势参考

| 字段路径 | 参考精度 | 说明 |
| --- | --- | --- |
| `bp.sbp` | ±8–12 mmHg | 收缩压 (mmHg)；< 90 触低血压阈值；90–119 正常范围；120–129 正常高值（临界，建议复测）；130–139 1级偏高；≥ 140 2级偏高。联合判定：任一触低（SBP<90 或 DBP<60）优先判血压偏低；否则取 SBP/DBP 各自级别中较高者（就高不就低原则） |
| `bp.dbp` | ±8–12 mmHg | 舒张压 (mmHg)；< 60 触低血压阈值；60–79 理想范围；80–84 正常血压；85–89 正常高值（临界）；≥ 90 1级偏高。与 sbp 联合判定，取级别较高者 |
| `bp.confidence` | — | 血压估算置信度 0–1；建议 ≥ 0.3 时参考结果；< 0.15 时字段可能不返回 |
| `bp.map_est` | ±8–12 mmHg | 平均动脉压 (mmHg)；正常 70–100；< 60 为休克阈值（需关注）；> 110 提示高灌注压 |
| `bp.pp_est` | ±8–12 mmHg | 脉压差 (mmHg)；正常 30–50；> 60 提示动脉硬化/主动脉关闭不全；< 25 提示低血容量/心功能不全 |
| `bp.features` | — | 脉搏波形特征 (ri, si, b_a, t_ratio) |

## `psych` 心理综合评分

**计费/模块:** 心理综合评分包 · 30,000 Token/次

10 项心理评估指标，均为 0–100 分。分数区间：85+ 优秀；75–85 良好；65–75 均值区间；60–65 略低；50–60 明显偏低（仅高 SQI 时出现）。另有 4 项行为衍生指标位于 behavior 模块。各维度参考误差约 ±10%。

| 字段路径 | 参考精度 | 说明 |
| --- | --- | --- |
| `psych.ms` | ±10% | 心理压力 (Stress)；HR 高+SDNN 低→低分；80+ 压力感低；< 60 高压力 |
| `psych.mr` | ±10% | 心理放松 (Relaxation)；Poincaré SD1/SD2 比率（副交感短期/长期比）；80+ 深度放松 |
| `psych.mf` | ±10% | 心理疲劳 (Fatigue)；HR/TP 倒置，TP 低+HR 高→低分；80+ 精力充沛 |
| `psych.sq` | ±10% | 睡眠质量 (Sleep Quality)；综合频域与几何特征；LF 或 SD2 偏低则下降 |
| `psych.mh` | ±10% | 心理健康 (Health)；Poincaré 面积大=调节空间宽；反映长期心理韧性 |
| `psych.mb` | ±10% | 心理平衡 (Balance)；交感/副交感比率偏离 1:1 的程度；80+ 高度平衡 |
| `psych.con` | ±10% | 专注力 (Concentration)；认知负荷高时 HR↑、HRV↓、呼吸率变化；80+ 高度专注 |
| `psych.mab` | ±10% | 记忆/自主神经基线 (Memory)；反映神经"年龄效应"，年轻健康基线更高 |
| `psych.adp` | ±10% | 适应能力 (Adaptation)；SDNN 与 RMSSD 协同；某项偏低则恢复速度受限 |
| `psych.sra` | ±10% | 抗压能力 (Resilience)；保持正常 HR 的同时保留 HRV 储备；80+ 强压下仍能调节 |

## `emotion` 情绪分析

**计费/模块:** 表层基础包 5k · 深层情绪包 20k Token

surface（面部表情，可伪装）vs deep（面部+rPPG 融合，抗伪装）。面试/测谎等场景推荐用 deep；UX 测试等场景用 surface。识别准确率约 75%（surface 与 deep 相同）。

| 字段路径 | 参考精度 | 说明 |
| --- | --- | --- |
| `emotion.surface.*` | 准确率 75% | 8 类表层情绪概率分布，总和为 1：Anger / Contempt / Disgust / Fear / Happiness / Neutral / Sadness / Surprise |
| `emotion.deep.*` | 准确率 75% | 8 类深层情绪（字段名同上）；多模态融合，抗表演性干扰，更贴近真实内心状态 |
| `emotion.dominant` | 准确率 75% | 主导情绪名称（8 类中概率最高者） |
| `emotion.dominant_confidence` | — | 主导情绪置信度；≥ 0.5 可信主导；0.25–0.5 倾向不明确；< 0.25 情绪复杂或面部信息不足 |

## `face_au` 面部动作单元

**计费/模块:** 面部 AU 包 · 20,000 Token/次

20 个融合 AU，值域 0–1。通用激活判断：0–0.15 未激活；0.15–0.40 轻度；0.40–0.65 中度（通常可见表情）；0.65–1.0 强烈激活。多源融合注视方向请用 behavior.gaze_yaw/gaze_pitch，精度更高。识别准确率约 90%。

| 字段路径 | 参考精度 | 说明 |
| --- | --- | --- |
| `brow_furrow` | 准确率 90% | 皱眉；> 0.4 明显；愤怒（双侧）或专注（单侧温和） |
| `brow_inner_raise` | 准确率 90% | 内眉上挑；> 0.3 悲伤眉（与 sadness_brow 联动） |
| `brow_outer_raise` | 准确率 90% | 外眉上挑；> 0.3 惊讶；< 0.1 且 brow_furrow 高 = 愤怒 |
| `cheek_puff` | 准确率 90% | 面颊鼓张；> 0.4 憋气或刻意鼓腮 |
| `cheek_squint` | 准确率 90% | 眯眼挤压；> 0.3 杜兴氏眼角收缩（真笑标志） |
| `eye_blink` | 准确率 90% | 眨眼闭合度；> 0.35 开始眨眼；> 0.8 完全闭合（PERCLOS 用 0.8 阈值） |
| `eye_gaze_h` | 准确率 90% | 水平注视（单源）；正值=右；负值=左 |
| `eye_gaze_v` | 准确率 90% | 垂直注视（单源）；正值=上；负值=下 |
| `eye_squint` | 准确率 90% | 眯眼程度；专注 / 真笑 / 强光刺激 |
| `eye_wide` | 准确率 90% | 瞪眼；> 0.4 恐惧或惊讶 |
| `jaw_open` | 准确率 90% | 下颌张开；> 0.3 说话；> 0.6 大张口；联合 mouth_openness 判断说话 |
| `jaw_lateral` | 准确率 90% | 下颌侧移；侧向咀嚼或说话偏移 |
| `mouth_smile` | 准确率 90% | 嘴角上扬；> 0.4 社交笑；配合 cheek_squint 判断是否为真笑 |
| `mouth_smile_asym` | 准确率 90% | 嘴角不对称；> 0.15 轻蔑或单侧面神经受限 |
| `mouth_frown` | 准确率 90% | 嘴角下垂；悲伤或厌恶 |
| `mouth_protrude` | 准确率 90% | 嘴唇前突；嘟嘴、亲吻 |
| `mouth_stretch` | 准确率 90% | 嘴唇横拉；咧嘴、恐惧 |
| `upper_lip_raise` | 准确率 90% | 上唇上抬；> 0.3 厌恶 |
| `nose_sneer` | 准确率 90% | 鼻翼皱缩；> 0.3 厌恶或愤怒（与 brow_furrow 叠加 = aggression） |
| `tongue_out` | 准确率 90% | 舌头外露；> 0.5 明确伸舌 |

<a id="behavior"></a>

## `behavior` 行为意图指标

**计费/模块:** 眼动意图包 20k · 心理包 30k · 健康包 30k Token

眼动意图 17 项 + 心理综合评分包衍生 4 项 + 面部健康包衍生 2 项（附赠不计费）。眼动意图类指标识别准确率约 80%。

| 字段路径 | 参考精度 | 说明 |
| --- | --- | --- |
| `gaze_yaw` | 准确率 80% | 水平视线偏移（多源融合）；正值=向右；负值=向左；\|值\| > 1 明显偏转 |
| `gaze_pitch` | 准确率 80% | 垂直视线偏移（多源融合）；正值=向上；负值=向下（如低头看手机） |
| `eye_aperture` | 准确率 80% | 眼睑闭合度 0–1；0=闭眼；1=正常睁眼；> 1.0 瞪眼（惊恐/恐惧） |
| `perclos` | 准确率 80% | PERCLOS 疲劳指数 0–1；> 0.15 驾驶监控级疲劳阈值（SAE J2057）；> 0.30 严重嗜睡 |
| `blink_rate` | 准确率 80% | 眨眼频率 (次/分)；正常 8–30；< 8 过度专注或疲劳；> 30 紧张/刺激 |
| `blink_close_ms` | 准确率 80% | 闭眼速度 (ms)；正常 70–100；> 150 眼睑沉重 |
| `blink_dwell_ms` | 准确率 80% | 闭合停留 (ms)；正常 50–150；> 200 疲劳/嗜睡指标（Caffier 2003） |
| `blink_open_ms` | 准确率 80% | 睁眼速度 (ms)；正常 70–100；> 150 眼睑沉重 |
| `gaze_stability` | 准确率 80% | 注视稳定性 0–1；> 0.7 专注凝视；< 0.3 视线游移、注意力分散 |
| `mouth_openness` | 准确率 80% | 嘴部开合度 0–1；> 0.15 说话中；> 0.5 大张口（惊讶/打哈欠） |
| `talking` | 准确率 80% | 说话状态（无需音频）；基于嘴部动作方差 |
| `clenching` | 准确率 80% | 咬牙紧张度 0–1；> 0.4 明显咬紧（紧张/疼痛） |
| `duchenne` | 准确率 80% | 真笑检测 0–1；> 0.3 真实笑容；< 0.1 且 smile 高 = 社交假笑 |
| `sadness_brow` | 准确率 80% | 悲伤眉；正值=内眉抬起+外眉下垂（八字眉）；负值=蹙眉 |
| `aggression` | 准确率 80% | 攻击性 0–1；> 0.5 皱眉+鼻皱叠加，强攻击性指征 |
| `confusion` | 准确率 80% | 疑惑/思考 0–1；> 0.2 单侧眉毛不对称抬起 |

## `liveness` 活体检测

**计费/模块:** 面部表层基础包

判断真实人脸 vs 照片/视频/屏幕翻拍攻击。rppg_sqi 是最强反欺骗信号。金融/身份认证建议 liveness_score ≥ 0.8；普通签到 ≥ 0.5 即可。光线过暗、人脸 < 20%、戴口罩/墨镜均会降低得分。参考准确率约 80%（标准拍摄条件）。

| 字段路径 | 参考精度 | 说明 |
| --- | --- | --- |
